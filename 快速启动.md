# 小说转剧本工具 v1.1

## 快速启动

### 1. 启动后端服务
```bash
cd novel-to-script
mvn spring-boot:run
```
后端会在 http://localhost:8080 启动

### 2. 启动前端服务（新开一个终端）
```bash
cd novel-to-script/frontend
npm install    # 首次运行需要安装依赖
npm run dev
```
前端会在 http://localhost:5173 启动

### 3. 使用方法
1. 打开浏览器访问 http://localhost:5173
2. 上传 TXT 格式的小说文件（或拖拽文件）
3. 点击「开始转换」按钮
4. 等待转换完成后查看和复制剧本

## 功能特点

- 📁 支持拖拽上传 TXT 文件
- 📝 实时预览小说内容
- 🔄 实时转换进度显示
- 👥 自动识别人物列表
- 🎬 场景化剧本输出
- 📋 一键复制剧本
- 🎨 简洁美观的界面

## 项目结构

```

├── src/                          # 后端源代码 (Java)
```

## 技术栈

- **后端**: Java 17, Spring Boot 3.x
- **前端**: Vue 3, Vite
- **UI**: 原生 CSS (无依赖)

## API 接口

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /api/v1/convert | 转换小说为剧本 |
| GET | /api/v1/health | 健康检查 |
